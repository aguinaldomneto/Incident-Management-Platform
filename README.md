# Incident Management Platform

Projeto desenvolvido para simular o gerenciamento de incidentes em ambientes de produção, utilizando Node.js, SQLite e Docker.

O projeto possui pipeline de Integração Contínua (CI) através do GitHub Actions, realizando automaticamente a validação da aplicação e o build da imagem Docker a cada alteração enviada para o repositório.

## Tecnologias

* Node.js
* Express
* SQLite
* Docker
* GitHub Actions
* Kubernetes

## Funcionalidades

* Health Check
* Cadastro de Incidentes
* Consulta de Incidentes
* Atualização de Status
* Logging de Requisições

## Objetivo

Simular um ambiente corporativo de operação e sustentação, permitindo o gerenciamento de incidentes e aplicação de boas práticas de observabilidade e automação.
