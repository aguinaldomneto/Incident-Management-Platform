# Incident Management Platform

Projeto desenvolvido para demonstrar conceitos de automação, observabilidade, gestão de incidentes e práticas DevOps.

## Tecnologias

* Node.js
* Express
* SQLite
* Docker
* GitHub Actions
* Kubernetes

## Funcionalidades

* Health Check
* Cadastro de Incidentes
* Consulta de Incidentes
* Atualização de Status
* Logging de Requisições

## Objetivo

Simular um ambiente corporativo de operação e sustentação, permitindo o gerenciamento de incidentes e aplicação de boas práticas de observabilidade e automação.
